# De Onthulling — Level 2

GitHub Pages-klare website.

## Online zetten
1. Maak een nieuwe GitHub-repository.
2. Upload alle bestanden en mappen uit deze ZIP naar de hoofdmap van de repository.
3. Ga naar **Settings → Pages**.
4. Kies bij **Build and deployment**: **Deploy from a branch**.
5. Selecteer branch **main** en map **/(root)**.
6. Sla op. GitHub toont daarna de openbare URL.

## Inhoud
- Laadscherm
- Countdown 3–2–1
- BOOM-effect
- Rotterdam-fotosequentie, gesynchroniseerd met de meegeleverde muziek
- Kort zwart beeld
- Goud vuurwerk
- Tweede BOOM
- Tekst: **WEES VOORBEREID OP LEVEL 3**

Er zijn geen externe libraries of internetbronnen nodig.
