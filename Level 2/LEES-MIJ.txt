DE ONTHULLING — LEVEL 2

UPLOAD NAAR GITHUB:
1. Open de hoofdmap van je repository De-Onthulling.
2. Kies Add file > Upload files.
3. Sleep de volledige map 'level-2' naar GitHub.
4. Controleer vóór het committen dat GitHub deze paden toont:
   level-2/index.html
   level-2/scene-1.jpg t/m scene-8.jpg
   level-2/rotterdam-level-2.m4a
5. Klik op Commit changes.

Daarna staat Level 2 op:
https://cynthiaschoonbrood-nld.github.io/De-Onthulling/level-2/

Level 1 in de hoofdmap blijft hierdoor behouden.
